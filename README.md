# The Book of Everything（万物之书）

基于中国四大名著及各类经典文学作品的 2D 游戏项目，使用 Godot 4.7 引擎开发。

## 游戏概述

玩家可以在三种独立模式中体验名著角色的魅力：

### 剧情模式
忠于原著的交互式动画演出。以"书"为单位，从书架选择名著，按章节体验原著剧情。核心体验是骨骼动画驱动的角色表演、镜头运镜、对话演出，部分关键剧情节点会穿插战斗。通关可解锁角色卡牌和升阶素材。

### 自走棋模式（PVP / PVE）
独立于剧情的策略竞技模式。将名著角色放置在棋盘上自动战斗，通过阵容搭配和羁绊系统（如"取经四人组"、"五虎将"）获得增益。

### 卡牌对战模式（PVP / PVE）
类似魔灵召唤的回合制卡牌对战。每张卡牌代表一个名著角色，拥有星级、属性、技能组和成长阶段。通过抽卡获取角色，编成队伍进行对战。

## 收录名著

- **西游记** — 孙悟空、猪八戒、沙悟净、唐僧等
- **三国演义** — 刘备、关羽、张飞、诸葛亮、曹操等
- **水浒传** — 宋江、武松、林冲、鲁智深等
- **红楼梦** — 贾宝玉、林黛玉、薛宝钗等
- 后续可扩展更多名著

## 技术栈

- **引擎**：Godot 4.7
- **语言**：GDScript
- **维度**：2D
- **渲染**：Forward Plus
- **物理**：Jolt Physics

## 目录结构

```
res://
├── src/                    # 脚本代码
│   ├── main/               # 主入口、全局管理器（Autoload）
│   ├── core/               # 共享核心系统（角色、卡牌、背包、经济、存档、动画）
│   ├── network/            # 网络/多人系统（预留）
│   ├── story/              # 剧情模式（动画演出、对话、镜头）
│   ├── autochess/          # 自走棋模式（棋盘、棋子、羁绊）
│   ├── card_battle/        # 卡牌对战模式（对战、卡池、竞技场）
│   ├── ui/                 # UI 系统
│   └── utils/              # 工具类
│
├── assets/                 # 美术/音频资源
│   ├── books/              # 按名著组织（角色素材、场景背景、道具）
│   ├── ui/                 # UI 通用素材
│   ├── effects/            # 特效
│   ├── audio/              # 音频（BGM/SFX）
│   └── chess/              # 自走棋棋盘素材
│
└── data/                   # 游戏数据配置
    ├── cards/              # 卡牌数据（.tres）
    ├── stories/            # 剧情脚本（JSON）
    ├── synergies/          # 羁绊配置
    ├── economy/            # 经济配置
    ├── growth/             # 成长数值表
    └── balance/            # 战斗数值平衡
```

## Autoload 单例

| 名称 | 脚本 | 职责 |
|------|------|------|
| EventBus | `src/core/event_bus.gd` | 全局信号总线，解耦模块通信 |
| SceneManager | `src/main/scene_manager.gd` | 场景切换管理 |
| GameManager | `src/main/game_manager.gd` | 全局游戏状态、模式切换入口 |

## 命名约定

- **目录/文件名**：小写下划线（`snake_case`），如 `journey_to_the_west/`
- **中文资源文件名**：可保留中文（如 `石猴.png`），Godot 原生支持
- **GDScript 类名**：PascalCase（如 `CardData`、`SkillData`）
- **数据格式**：设计数据用 `.tres`（编辑器可编辑）；剧情脚本用 JSON；存档用 JSON
