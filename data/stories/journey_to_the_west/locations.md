# 西游记前10回 — 场地/场景一览（以场地为维度）

- **四大部洲**
  - 第一回 — 东胜神洲、西牛贺洲、南赡部洲、北俱芦洲（开篇设定）
- **傲来国**
  - 第一回 — 东胜神洲海外一国，国近大海
- **傲来国城池**
  - 第三回 — 悟空弄风吹乱，搬光兵器馆/武库
- **花果山**
  - 第一回 — 傲来国海中名山，十洲之祖脉，三岛之来龙。悟空诞生地
  - 第二回 — 悟空故乡，被混世魔王侵扰后悟空收复
  - 第三回 — 悟空领群猴操演、结义、宴饮
  - 第四回 — 悟空两次返回
  - 第五回 — 悟空回到花果山，天兵围困
  - 第六回 — 战场，被天罗地网围困
- **水帘洞**
  - 第一回 — 花果山瀑布之后，内有石碣"花果山福地，水帘洞洞天"。悟空与众猴居所
  - 第二回 — 被混世魔王侵扰
  - 第三回 — 桥下水通东海龙宫
  - 第四回 — 悟空洞府
  - 第五回 — 悟空与众猴躲藏
  - 第六回 — 悟空洞府
- **铁板桥**
  - 第三回 — 水帘洞铁板桥，桥下水通东海龙宫
  - 第五回 — 四健将与九曜星交战处
- **南赡部洲**
  - 第一回 — 悟空渡海到南赡部洲寻仙问道，串长城、游小县八九年
- **西牛贺洲**
  - 第一回 — 再渡西洋大海至西牛贺洲寻仙
- **灵台方寸山**
  - 第一回 — 西牛贺洲灵台方寸山。悟空寻到菩提祖师之处。山有石碑"灵台方寸山，斜月三星洞"
- **斜月三星洞**
  - 第一回 — 灵台方寸山中的洞府，须菩提祖师居所。内有瑶台、深阁琼楼、珠宫贝阙
  - 第二回 — 菩提祖师讲道传法之所，悟空在此修行约二十年
- **烂桃山**
  - 第二回 — 斜月三星洞后山，悟空在此吃了七次饱桃（七年）
- **东胜神洲**
  - 第二回 — 悟空回程
- **水脏洞**
  - 第二回 — 直北下，混世魔王居所。悟空闯入放火烧毁
- **东海（东洋大海）**
  - 第三回 — 悟空入水寻龙宫
- **东海龙宫（水晶宫）**
  - 第三回 — 东海龙王敖广宫殿。悟空取走金箍棒得披挂
- **幽冥界**
  - 第三回 — 城门有铁牌"幽冥界"
- **森罗殿**
  - 第三回 — 十代冥王审案之所，悟空在此勾销猴属名号
- **翠云宫**

  - 第六回 — 玉帝与观音议兵
  - 第七回 — 悟空打到灵霄殿外，如来前来救驾
- **通明殿**
  - 第三回 — 灵霄殿前，大臣等候之处
  - 第五回 — 悟空假传旨骗赤脚大仙此处
  - 第六回 — 众仙聚议
  - 第七回 — 悟空打入通明殿

  - 第三回 — 天庭南门
